Hello World!!

テスト投稿です。

このブログは静的サイトジェネレータHexoを使用しています。

技術的な内容を記載していきたいと思います。

Static Site Generatorsを見ると
https://staticsitegenerators.net/

人気ランキング
1位 Jekyll
2位 Octopress

とRuby製の静的サイトジェネレータが人気ですが、Rubyのバージョン違いなどで、エラーがよく発生します。

そこで、人気ランキング4位のJavaScript製HEXOを利用してみたいと思います。
CentOSにインストールしています。

npmインストール

sudo yum -y install nodejs npm –enablerepo=epel

hexoインストール

npm install -g hexo

サイト生成

vagrantにインストールしていますので、ホストから確認できるアドレスを指定します。

hexo init blog
cd blog
npm install
hexo server -i 192.168.33.10

確認

http://192.168.33.10:4000/

あっという間にブログができたのではないでしょうか！！

直感的にJeykyllよりビルドが早くて軽い感じがします。

テーマ変更

好きなテーマを選びます。
https://github.com/hexojs/hexo/wiki/Themes

git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia
//_config.yml修正
theme: yilia
hexo server -i 192.168.33.10

簡単なので是非試してみて下さい。

みたいな感じで書いていきます。
